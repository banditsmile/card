经销商提成级数 不限制的话，填 -1 即可 [?]

-1 ：表示不限制，所有上下级，只要有差价，就可以获得提成
0 ：所有级别都不给提成，不管有没有差价
1:一级提成(上下级)，只有直属上级有提成
2:二级提成(上中下级)，上中下级关系的时候，上中有提成
3:三级提成(abcd级)，abcd级的时候，abc有提成
依此类推，建议填写-1即可，不限制
